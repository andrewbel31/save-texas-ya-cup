# Приложение Save Texas для Yandex CUP 2021

## Демо работы: https://youtu.be/OZ7dMZ_EW1s

### Как работает приложение?

В качестве облачного хранилища используется firebase realtime database. Она сама делает много вещей,
например офлайн хранение данных, синхронизация при появлении сети, синхронизация между клиентами.

Основной стек - MVI + RxJava2
